Tensorflow基础

  • Google Brain的第二代机器学习框架
  • 开源社区活跃
  • 可扩展性好：在分布式下支持得非常好，一机多卡，多机多卡
  • API健全，对用户友好
    ## 计算图模型
  • 命令式编程
>>> x=2
>>> y=3
>>> b=1
>>> x * y + b
>>> 7
  • 声明式编程
>>> import theano.tensor as T
>>> from theano import function
>>> x = T.dscalar('x')
>>> y = T.dscalar('y')
>>> b = T.constant(1)
>>> z = x * y + b
>>> f = function([x, y], z)
>>> f(2, 3)
>>> array(7.0)
  • 命令式编程与声明式编程比较
命令式编程 声明式编程
如何执行 立即执行 先构建计算图，再输入数值进行计算
优点 容易理解、灵活、精准控制、易于调试 先拿到计算图，便于优化。可视化计算图。方便存取
缺点 提供整体优化很难。实现统一的辅助函数很难 很多语言特性用不上，不易于调试；监视中间结果不简单

神经元分类

  • 神经元：是最小的神经网络

机器学习简介

机器学习应用举例

  • 分类问题：图像识别、垃圾邮件识别
  • 回归问题：股价预测、房价预测
  • 排序问题：点击率预估、推荐
  • 生成问题：图像生成、图像风格转换、图像文字描述生成

机器学习应用流程

训练数据-》定义问题-》选择算法-》模型-》线上服务-》训练数据

机器学习岗位要做什么

  • 数据处理（采集+去噪）
  • 模型训练（特征+模型）
  • 模型评估与优化（MSE、F1-score、AUC+调参）
  • 模型应用（A/B测试）
